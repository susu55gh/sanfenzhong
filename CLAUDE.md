# CLAUDE.md — 三分钟

"给自己3分钟"——极简心理小游戏集合，每次 3 分钟，用完即走。

## 定位

用户不是来"放松"的，是来**夺回3分钟的主动权**。在一整天被动响应工作和家庭之后，主动选择一件只属于自己的小事。

## 项目结构

```
3分钟/
├── index.html          # 主页面（所有页面状态在一个文件）
├── style.css           # 全局样式
├── app.js              # 主逻辑（页面切换、状态管理）
├── games/
│   ├── breathing.js    # 呼吸球（4-7-8 呼吸法）
│   ├── tear.js         # 撕掉这页（写烦恼→撕掉动画）
│   └── paint.js        # 涂满颜色（Canvas 涂色）
└── CLAUDE.md
```

## 设计系统

- 主背景：#F5F0EB（暖白/奶油色）
- 文字主色：#2C2C2C
- 文字辅色：#8C8C8C
- 强调色：#7C9A8B（鼠尾草绿）
- 完成/成功：#A8C5B0
- 圆角：卡片 16px，按钮 24px（胶囊形）
- 动画：ease-in-out，300ms 过渡，不做刺激性动画

## 防沉迷

- 每天最多 3 次
- 无积分、无排行、无打卡、无推送
- 5 分钟内重复打开 → "你刚休息过，去喝杯水吧"

## 数据存储

全部 localStorage，不上传任何数据。
